Module Monte

module Monte: sig .. end

An implementation of the Monte Carlo tree search algorithm, used for the hard difficulty AI.

type visits = float

The type of a visit counter.

type wins = float

The type of a win counter.

type tree =
| Leaf
| Node of State.t * (int * int) * visits Pervasives.ref
* wins Pervasives.ref * tree * tree list Pervasives.ref

The type of a tree of game states.

val cartesian_product : 'a list -> 'b list -> ('a * 'b) list

cartesian_product l1 l2 is a list of coordinates generated from the cartesian product of two lists of l1 and l2.

val (--) : int -> int -> int list

i -- j is the list of integers from i to j, inclusive.

val next_possible_moves : State.t -> (int * int) list

next_possible_moves gm is a list of possible coordinates the AI can move to given current game gm.

val next_possible_states : tree -> tree list

next_possible_states n is a list of the next possible states the game can transition to given the current state n.

val random_move : State.t -> int * int

random_move gm is a random coordinate for the AI to move given the current state of the game gm.

val get_children : tree -> tree list

get_children n is the list of children of node n. Requires: n is not Leaf.

val select : tree -> tree

select n is the child of node n with the highest upper confidence bound. Requires: n is not Leaf.

val expand : tree -> unit

expand n sets the children of node n to all possible subsequent moves. Requires: n is not Leaf.

val get_game : tree -> State.t

get_game n is the current game state at node n. Requires: n is not Leaf.

val simulation_node : tree -> tree

simulation_node n is the node that the AI runs the simulation on given root node n.

val simulate : tree -> Board.square

simulate n is the winner after running a random simulated game on node n. Requires: n is not Leaf.

val update_win : float Pervasives.ref -> unit

update_win w updates the winning score.

val update_visit : float Pervasives.ref -> unit

update_visit v updates the number of visits.

val backpropagate : tree -> Board.square -> unit

backpropagate n sq updates the winning score and number of visits for each node given node n and winning player sq. Requires: n is not Leaf.

val next_coords : tree -> int * int

next_coords n is the coordinate of the move to reach the state in node n from the state in the parent node of n. Requires: n is not Leaf.

val next_move : State.t -> int * int

next_move () is the next optimal move for the AI.
