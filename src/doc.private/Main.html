Module Main

module Main: sig .. end

The entry point for the game.

val ideal_width : int

ideal_width is the smallest terminal window width at which a user can play comfortably.

val ideal_height : int

ideal_height is the smallest terminal window height at which a user can play comfortably.

val resize_window : unit -> unit

If a user has window dimensions smaller than ideal_width * ideal_height, adjusts terminal dimensions to match the ideal dimensions.

val quit_game : unit -> 'a

Prints an exit message and exits the program.

val prompt_status : unit -> Play.sub_selection

Prompts the user for whether they want to host a game or join a game. Only called if the user chose to play against another player.

val prompt_difficulty : unit -> Play.sub_selection

Prompts the user for a difficulty setting. Only called if the user chose to play against the computer.

val prompt_mode : unit -> Play.playmode * Play.sub_selection

Prompts the user for a playing mode.

val prompt_player_name : unit -> string

Prompts the user for a name.

val make_player : Play.playmode -> string

Creates a player for the score board from the player name.

val prompt_style : unit -> Play.style * (Play.playmode * Play.sub_selection)

Prompts the user for the style of game.

val prompt_address : unit -> Unix.inet_addr

Prompts the user for an IP address.

val prompt_start : unit -> unit

Prompts the user to start the game.

val main : unit -> unit

Launches the main menu. Credit to http://patorjk.com/software/taag for being an inspiration for the text graphics.

main () starts the game after prompting the player(s).
