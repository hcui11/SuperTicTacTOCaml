Module Play

module Play: sig .. end

Runs the game logic.

type playmode =
| VSPlayer
| VSComputer
| Test

The type of game modes.

type sub_selection =
| Easy
| Hard
| Host
| Join

The type of AI difficulty or the client-server status of the player.

type style =
| Normal
| Blitz

The type of game style.

val max_socket_data_length : int

max_socket_data_length is the maximum allowed length of data transmitted socket communication.

val timer : Unix.interval_timer

timer is the countdown timer loaded at the start of the turn.

val max_time : float
val time_left_X : float Pervasives.ref

time_left_X is the amount of time player X has left.

val time_left_O : float Pervasives.ref

time_left_O is the amount of time player O has left.

val set_time_stat : State.stat list -> unit
val quit_game : unit -> 'a

Prints an exit message and exits the program.

val set_time : Board.square -> Unix.interval_timer_status

set_time p sets the countdown timer to the amount of time player p has left.

val set_buffer_time : unit -> Unix.interval_timer_status

set_buffer_time () sets the countdown timer to a time buffer between rounds.

val update_time : Board.square -> unit

update_time p updates the amount of time player p has left.

val handle : 'a -> 'b

handle _ prints the winning message and exits the game.

val signal_handle : (int -> unit) -> unit

signal_handle () determines how to handle a signal once a player has no time left.

val reset_time : unit -> unit

reset_time () resets the player times for a new game.

val perform_AI_move : State.t -> sub_selection -> bool -> unit

perform_AI_move executes a single AI move on game state t. The AI strategy is determined by the difficulty level sub_select.

val send_data : Unix.file_descr -> string -> unit

send_data sock data sends data to socket sock.

val receive_data : Unix.file_descr -> string

receive_data sock is the data received from socket sock. Can have a maximum length of max_socket_data_length.

val execute : State.t ->
       Unix.file_descr ->
       Unix.file_descr ->
       style -> playmode -> sub_selection -> bool -> Board.square

execute st m diff print executes player commands on game state st with game mode m. If m = VSComputer, diff represents the difficulty of the computer AI. Prints the board grid after every move if print is true.
