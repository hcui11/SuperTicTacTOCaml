Module Board

module Board: sig .. end

Governs the game board of an Ultimate Tic Tac TOCaml game.

type square =
| X
| O
| Blank

The type of squares.

val other_square : square -> square

other_square sq returns the opposite square of sq. Requires: sq cannot be Blank.

type board = square array

The type of a 3x3 Tic-Tac-Toe board.

type game = board array

The type of an Ultimate Tic-Tac-Toe game.

val board_side_length : int

The side length of the board.

val num : int

The number of squares in a single sub-board, equal to board_side_length * board_side_length.

val blank_board : unit -> square array

The initialized sub-board full of blanks.

val initial_game : unit -> square array array

The initialized game full of blanks.

val index_of_coordinate : int -> int -> int

index_of_coordinate x y is the index of the coordinate (x,y) in a game.

val board_of_coordinate : int -> int -> int

board_of_coordinate x y is the board that (x, y) is located in.

val set_game_square : 'a array array -> int -> int -> int -> 'a -> unit

set_game_square t x y curr input sets the square at coordinate (x,y) in a board at index curr in game t to the given input.

val get_game_square : 'a array array -> int -> int -> int -> 'a

get_game_square t x y curr gets the square at coordinate (x,y) in a board at index curr in game t.

val get_board : 'a array -> int -> 'a

get_board gm i gets the board at index i in game gm.

val is_full : square array -> bool

is_full bd returns true if the board bd contain Blank.

val set_board_square : 'a array -> int -> 'a -> unit

set_board_square bd i in sets the square at index i in a board bd to value in

val get_board_square : 'a array -> int -> 'a

get_board_square bd i gets the square at index i in a board bd.

val check_win : square array -> bool

check_win bd returns true if one player has won the board bd.

val tie_breaker : square array -> square

tie_breaker bd breaks ties if bd is full and has no winner. Returns the square value with the majority on the board bd.

Requires: is_full bd = true && check_win bd = false

val string_of_square : square -> string

string_of_square s is a string representation of a square s.

val def : ANSITerminal.style list

The default printed text style.

val sty : ANSITerminal.style list

The highlighted printed text style.

val print_column_labels : unit -> unit

print_column_labels prints the column indices for the grid.

val print_horizontal_div : ANSITerminal.style list -> unit

print_horizontal_div h prints the horizontal line between grid rows.

val print_mini_div : unit -> unit

Prints a small board divider.

val mini_board_string : board -> string array

Prints one row of the miniboard bd.

val print_grid : square array array -> int -> board -> unit

print_grid g curr mini prints game grid g with currently selected board curr along with associated mini-board mini to the console.
